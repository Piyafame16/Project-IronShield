# 📊 IronShield Dashboard Overview

ไฟล์นี้เป็น Tableau Packaged Workbook (.twbx) ที่ใช้ในการวิเคราะห์และตัดสินใจเชิงกลยุทธ์:

### **Key Views:**
1. **Threshold Optimizer:** วิเคราะห์จุดคุ้มทุนระหว่างการตรวจจับโจร (Recall) และการรบกวนลูกค้า (FPR)
2. **Detection Speed Matrix:** วัดประสิทธิภาพความเร็วในการระงับบัญชี (Time to Detect) รายบัญชี
3. **Mule Behavioral Analytics:** เจาะลึกพฤติกรรมการโอนเงินแบบ Pass-through (< 60 นาที)

**Note:** หากไม่มีโปรแกรม Tableau Desktop สามารถดูสรุปผลการวิเคราะห์ได้ในไฟล์ PDF ในโฟลเดอร์ `/documents`
